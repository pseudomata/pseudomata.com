Software

Mostly software that I use on a daily (or atleast semi-regular) basis. Some things change while others stick around for years and I'm especially grateful for certain software that I've been with for many years (decades, even) like Emacs and Linux.

  • macOS, NixOS, FreeBSD, and Windows 11 (yuck, but gaming)
  • Nix for package management (replacing Homebrew on macOS)
  • Text Editors
    • Emacs for most code
    • Zed (a better and faster VS Code)

  • Arc browser (I use Firefox and Safari too and might switch entirely to Safari)
  • Apple Notes for taking quick notes
  • Ulysses writing app for long-form writing
  • Muse app for research
  • Element app for Matrix chats
  • Alfred app (considering Raycast instead)

Services

  • 1Password is by far the best designed password manager, though I'm sad they ditched their native application in favor of an Electron app
  • Mullvad VPN
  • My devices are configured to use Quad9 DNS (it's free, though I happily donate)
  • Tailscale is exceptional
  • Proton for mail, likely migrating to Fastmail
  • iCloud+ for storage
  • Cloudflare for domains (considering migrating to DNSimple)
  • Fly.io has become my service-of-choice for running applications in the cloud
  • GitHub for hosting code
  • Buildkite is by far the best CI/CD tool I've used (CircleCI is a great fully-hosted option)
