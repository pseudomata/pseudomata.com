Welcome to my tiny corner of the internet! I'm an independent researcher and software engineer. I like to think of myself as a life-long computing enthusiast. My interests include programming languages, infrastructure, distributed & decentralized systems, cryptography, and design.

This website is currently a work in progress, but everything you see here is hand-crafted HTML and CSS. I'm working on a custom static-site generator in Haskell using Pandoc. My goal is to start from scratch, maintain this minimal aesthetic, and slowly build bespoke tools to generate content for my website (i.e. generating static HTML files from Markdown).

I'm also working on a simple key discovery and distribution mechanism for digital signatures using DNS and researching Decentralized Identifiers (DID) to see if they can be useful in this endeavor.
