Bonjour! This is my tiny corner of the internet. I'm an independent researcher, software engineer, and a life-long computing enthusiast. My interests include programming languages, infrastructure, decentralized systems, cryptography, and design.

This website is currently a work in progress, but everything you see here is hand-crafted HTML and CSS. I'm working on a custom static-site generator in Haskell using Pandoc. Part of my experimentation is using tree-sitter to generate syntax highlighing for code blocks.

I'm also working on a simple public-key discovery and distribution mechanism for digital signatures using DNS and researching Decentralized Identifiers (DID) to see if they can be useful in this endeavor.
